# offset

百度笔试题：

请编写宏，计算结构体中某变狼相对于首地址的偏移，并给出说明；

官方宏：
```C
#include <stdio.h>
#include <stddef.h>

typedef struct Student
{
	char c1;
	int a;
	char c2;
}S;//所占空间为12个字节

int main()
{
	S s = { 0 };
	printf("%d\n", offsetof(S, c1));
	printf("%d\n", offsetof(S, a));
	printf("%d\n", offsetof(S, c2));
	return 0;
}

·

```
模拟实现：

```C
#define OFFSETOF(struct_name,struct_member) (int)&(((struct_name*)0)->struct_member)
```

解析：

先将地址0强制转化为一个结构体指针

找到这个结构体指针中结构体成员的位置

再将这个结构体成员取地址

再将这个地址强制转化为一个整型

由此得到了这个结构体成员的位置
