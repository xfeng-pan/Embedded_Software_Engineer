# 动态增长的通讯录

默认可以存放3个人的信息
当防线当前通讯录满的时候，我们进行扩容，每次增加两条信息的空间

所以空间的大小：3，5，7，9，11......

# 指针类型的动态通讯录

## 改造通讯录的类型
除了一个指向PeopInfo类型结构体的指针
还需要显示当前存了几个人的信息
还需要显示当前通讯录的最大容量


```C
//通讯录类型
struct Contact
{
	struct PeoInfo *data;//定义一个指向PeopInfo类型结构体的指针
	int size;//记录当前已经有多少人的信息
	int capacity;//记录当前通讯录的容量
};
```
